üëã Welcome to my Home Operations repository, **tnwks** /tee-networks/. This is a mono repository for my home infrastructure that has a primary goal of housing the IaC for every single piece that is possible.  I try and build my infrastructure as close to a best-practice, modern, production environment as possible while keeping costs relatively low.

 I try to adhere to Infrastructure as Code (IaC) and GitOps practices using the tools like [Ansible](https://www.ansible.com/), [Terraform](https://www.terraform.io/), [Kubernetes](https://kubernetes.io/), [Flux](https://github.com/fluxcd/flux2), [Renovate](https://github.com/renovatebot/renovate) and [GitHub Actions](https://github.com/features/actions).
